🚀 STEP 1: START THE APPLICATION
══════════════════════════════════

Windows:
--------
1. Open PowerShell or Command Prompt
2. Navigate to: e:\web development
3. Run command:
   npm start

4. You should see:
   "LearnHub server running on http://localhost:5000"

Mac/Linux:
----------
1. Open Terminal
2. Navigate to: cd /path/to/web\ development
3. Run command:
   npm start

4. You should see:
   "LearnHub server running on http://localhost:5000"

🌐 STEP 2: OPEN IN BROWSER
════════════════════════════

Windows:
--------
Option A: Using File Explorer
  1. Navigate to: e:\web development
  2. Double-click: app.html
  3. Your default browser opens

Option B: Manually
  1. Open any web browser (Chrome, Firefox, Edge, Safari)
  2. Address bar: type app.html path OR
  3. Press Ctrl+O and browse to app.html

Mac:
----
1. Open Finder
2. Navigate to your web development folder
3. Double-click: app.html
4. App opens in Safari or your default browser

Linux:
------
1. Open file manager
2. Navigate to web development folder
3. Right-click app.html
4. Open with → Your preferred browser

🐛 TROUBLESHOOTING
══════════════════

Problem: "npm start gives error"
Solution:
- Run: npm install
- Try: npm start again
- Check Node.js version: node -v (should be v14+)

Problem: "Images don't upload"
Solution:
- Check browser console (F12)
- Verify file is image format
- Try smaller image file
- Check uploads/ folder permissions

Problem: "XP not increasing"
Solution:
- Make sure backend is running
- Try refreshing page (F5)
- Check browser console (F12)
- Clear browser cache

Problem: "Q&A responses are generic"
Solution:
- This is expected - simulated AI
- Ask more specific questions
- Add more details to problem
- Try "Explain" questions

Problem: "Footer not showing"
Solution:
- It's at bottom right: scroll down
- Footer only shows when space available
- Resize window to see it
- Check for overlap with main content
