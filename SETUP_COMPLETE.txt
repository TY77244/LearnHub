❓ TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

PROBLEM: "Cannot upload file"
SOLUTION: 
• Check backend server is running (npm start)
• Verify supported file types (PDF, DOC, DOCX, etc.)
• Check browser console for errors (F12 key)
• Try with a smaller file first

PROBLEM: "Login fails"
SOLUTION:
• Verify you created an account first
• Check email and password are correct
• Ensure database exists (learnhub.db)
• Clear browser cookies

PROBLEM: "Documents not showing"
SOLUTION:
• Refresh page (F5 or Ctrl+R)
• Check if backend is running
• Look at browser console (F12)
• Verify you're logged in

PROBLEM: "Port 5000 already in use"
SOLUTION:
• Close other Node.js applications
• Or change PORT in app.js to 5001 or 5002
• Check what's using port: netstat -ano | findstr :5000 (Windows)

PROBLEM: "Python backend won't start"
SOLUTION:
• Install Python from python.org
• Run: pip install -r requirements.txt
• Make sure requirements.txt is in same folder
• Use: python3 structure.py (not python)
